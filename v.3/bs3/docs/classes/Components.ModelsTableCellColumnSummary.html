Components.ModelsTableCellColumnSummary Class

Extends Ember.Component
Defined in: addon/components/models-table/cell-column-summary.js:45

Component for table-footer cells. Used as column-summary.

It yields several properties:

  • minSelected - min of selected items
  • maxSelected - max of selected items
  • sumSelected - sum of selected items
  • avgSelected - average of selected items
  • medianSelected - median of selected items
  • minData - min of data
  • maxData - max of data
  • sumData - sum of data
  • avgData - average of data
  • medianData - median of data

Here selectedItems and data are bound from models-table and are mapped by column.propertyName.

Component should be used only for column with set propertyName.

Component should be extended or its template should be overridden.
